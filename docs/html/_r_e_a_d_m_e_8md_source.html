15 ## Overview
16 This is a controller module for a robot that uses an Ackermann Steering Model, This controller is to be implemented in a four wheeled Robot made by ACME Robotics.
17 
18 #### Description
19 A controller is build for a 4 wheeled robot with an Ackermann steering to navigate through its environment. The controller consist of a PID algorithm which ensures that the velocity converges to the set point, and a Steer Algorithm that helps the robot turn.
20 <br>The PID Algorithm is a control loop mechanism that calculates the error and applies correction through proportional, integral and derivative gains. The Steer Algorithm is developed to turn the robot, which is done by calculating the length of an arc inscribed between the current robot heading and target heading, the length when divided by the robot velocity gives the time for which the wheels need to be kept at angles given by Ackermann steering Model.
21 <br>The input to the controller will be provided by the perception model developed by the ACME Robotics.
22 <br>The Demonstration of the controller will be given by plotting a graph that shows convergence of velocity & Heading angle to the targets with respect to time.
23 
24 #### Features
25 * Velocity control during turning.
26 * Protection against Skidding, by limiting velocity during turning.
27 * Protection agains over volting the Motors.
28 
29 #### Application
30 * Mobile Wheeled Robots
